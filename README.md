# Snake Game

Play the classic game Snake on your Roku. Collect the Red food and grow as long as you can; make sure to avoid running into the edges of the screen, or your own tail!

Download and sideload the roku-deploy.zip file from /out/roku-deploy.zip on Roku, or build yourself from /dist/apps/snake-game
